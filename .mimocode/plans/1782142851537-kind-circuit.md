The `ia-tool` CLI currently shows a static ASCII logo with picocolors gradient. The user wants:

### Phase 2: Animated ASCII Logo (`src/utils/ascii.ts`)

Replace static logo with animated reveal using dui's `animate` function:

```typescript
import { animate, colors } from '@bdocs/dui'

const logoLines = [
  "  ___   _       _____ ___   ___  _     ",
  " |_ _| / \\     |_   _/ _ \\ / _ \\| |    ",
  "  | | / _ \\      | || | | | | | | |    ",
  "  | |/ ___ \\     | || |_| | |_| | |___ ",
  " |___/_/   \\_\\    |_| \\___/ \\___/|_____|",
]

const colorGradient = ['#d946ef', '#a855f7', '#3b82f6', '#06b6d4', '#06b6d4']

// Animate: each line appears with delay, using colors
export async function renderAsciiLogo(): Promise<void> {
  for (let i = 0; i < logoLines.length; i++) {
    // Use colorize from dui for hex colors
    const line = colorize(logoLines[i], colorGradient[i])
    console.log(line)
    await sleep(80) // 80ms delay between lines
  }
  // Add subtitle with dim style
  console.log(colors.dim(`\n  ⚡ IA-TOOL CLI v1.0.0 — AI Configs & Smart Git Tools\n`))
}
```

**Alternative: More advanced animation** using `animate()` for typing effect per line.
